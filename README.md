# ScreenGuide Landing Page

A simple, elegant landing page for the ScreenGuide app.

## Features

- Responsive design that works on all devices
- Clean, modern aesthetic with gradient background
- Smooth animations on load
- App logo and branding

## Deployment

This landing page can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Local Development

Simply open `index.html` in your browser to view the landing page locally.
